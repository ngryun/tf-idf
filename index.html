TF-IDF 계산기

여러 문서를 붙여 넣으면 자동으로 기본 불용어를 제거하고, TF, DF / IDF, TF-IDF 표를 한 번에 계산합니다.

TF

문서 안에서 단어가 등장한 비율

DF / IDF

현재 선택: IDF = 전체 문서 수 / 단어가 출현한 문서 수

TF-IDF

문서마다 핵심어를 뽑는 최종 점수

설정

기본 불용어는 자동 적용됩니다. 필요한 단어만 추가로 넣어 주세요.

IDF 계산 방식
기본 불용어 처리 기준 보기

조사, 접속사, 지시어 같은 자주 쓰이지만 정보량이 낮은 단어를 기본 제거합니다. 한국어 형태소 분석기 없이도 기본적인 수업 활동은 충분히 가능합니다.

문서별 제목을 적고 긴 글을 붙여 넣으세요.

문서별 핵심어

DF / IDF 테이블
